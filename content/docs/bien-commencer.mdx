Notre plateforme est conçue pour s’adapter à vos besoins, que vous soyez un utilisateur occasionnel ou une équipe technique souhaitant personnaliser son environnement. Voici les différentes façons d’y accéder et de l’utiliser :

## 1. Version web

Vous pouvez utiliser la version web de notre plateforme directement depuis votre navigateur, sans aucune installation.  
- **Accès immédiat** : Rendez-vous simplement sur notre site.
- **Compatibilité** : Fonctionne sur tous les navigateurs modernes.
- **Mises à jour automatiques** : Profitez toujours des dernières fonctionnalités.

## 2. Application dédiée

Pour une expérience plus fluide et intégrée, téléchargez notre application :
- **Disponible sur plusieurs systèmes** : Windows, macOS.
- **Performances optimisées** : Utilisation des ressources locales pour une meilleure réactivité.
- **Fonctionnalités supplémentaires** : Notifications natives, intégration système, etc.

## 3. Construire et déployer soi-même

Vous souhaitez aller plus loin ?  
Il est possible de construire et déployer votre propre instance de la plateforme :
- **Accès au code source** : Téléchargez le code depuis notre dépôt officiel.
- **Guide détaillé** : Suivez notre documentation pour développeurs étape par étape pour le build et le déploiement.
- **Personnalisation** : Adaptez la plateforme à vos besoins spécifiques.
- **Contrôle total** : Hébergez la solution sur votre propre infrastructure avec le contrôle total sur votre base de données.

---

En résumé, que vous choisissiez la version web, l’application ou le déploiement personnalisé, notre plateforme vous offre flexibilité et simplicité pour bien démarrer.
