### Préréquis

Pour tester EJDA Chat avec Vite, voici ce dont vous aurez besoin :

* **Unix/Linux/macOS** : ces systèmes sont recommandés car ils sont compatibles nativement avec la plupart des outils de développement web.
* **[WSL](https://learn.microsoft.com/fr-fr/windows/wsl/install) ou [Git](https://git-scm.com/)** (si vous êtes sur Windows) :  
  - **WSL** (Windows Subsystem for Linux) permet d'utiliser un environnement Linux sur Windows, ce qui facilite l'installation et l'utilisation des outils de développement.
  - **Git** est un gestionnaire de versions indispensable pour télécharger et gérer le code source.
* **Un fork d'EJDA Chat** : un fork est une copie du projet sur votre propre compte GitHub. Cela vous permet de travailler sur le code et de proposer des modifications sans affecter le projet original.
